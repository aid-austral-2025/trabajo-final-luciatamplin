# Trabajo Final - Lucía Tamplin

La base de datos utilizada corresponde a un estudio sobre el tiempo de pantalla de las personas participantes y su relación con distintos indicadores de salud mental y bienestar general.

Descripción de las variables de la base de datos:

**user_id**: Identificador único de cada participante.
**age**: Edad del participante (en años).
**gender**: Género del participante (Masculino, Femenino, No binario/Otro).
**occupation**: Ocupación del participante (por ejemplo: Estudiante, Empleado, Jubilado, etc).
**work_mode**: Modalidad de trabajo o estudio (Remoto, Presencial, Híbrido).
**screen_time_hours**: Horas totales de tiempo en pantalla por día.
**work_screen_hours**: Horas diarias frente a la pantalla destinadas a tareas laborales o académicas.
**leisure_screen_hours**: Horas diarias de uso recreativo de pantallas.
**sleep_hours**: Horas de sueño por noche.
**sleep_quality_1_5**: Calidad del sueño (escala de 1 = Muy mala a 5 = Excelente).
**stress_level_0_10**: Nivel de estrés (escala de 0 a 10).
**productivity_0_100**: Nivel de productividad (escala de 0 a 100).
**exercise_minutes_per_week**: Minutos de actividad física por semana.
**social_hours_per_week**: Horas semanales dedicadas a actividades sociales.
**mental_wellness_index_0_100**: Índice de salud mental (escala de 0 a 100).
